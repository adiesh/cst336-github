Machine Learning gives computers the opportunity to learn without being explicitly programmed/trained.

Through machine learning, one can create a model/algorithm to predict an outcome of a certain event.



How to make these predictions, we must do the following:

  1. Conduct research: Research on previous contests to help make the best assessments. Create a hypothesis.
  2. Collect the Data: Create scripts to pull some of the data from a given time frame.
  3. Model Selection: Build different models to determine the best performance. You will need a training and test data set. Depending on the accuracy of your data, choose which set is most accurate.
  4. Results: Analyze the results of your model. Determine if other features are necessary. Compare your results with other attempts.


With the above, we can create a model to try and predict the outcome of an NFL season or whatever event you are trying to predict.
